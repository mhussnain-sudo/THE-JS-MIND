# 02 - JavaScript Collections

## What are JavaScript Collections?

Collections are built-in data structures designed to store and manage groups of values efficiently.

Unlike Arrays and Objects, which are general-purpose structures, collections such as Set and Map solve specific problems like uniqueness, fast lookups, and key-value storage.

---

## Why learn Collections?

Many interview questions become much simpler when you recognize that a collection is the right tool.

For example:

- Finding duplicates
- Counting occurrences
- Fast lookups
- Storing unique values
- Mapping keys to values

Instead of writing complex loops, collections often provide a cleaner and more efficient solution.
