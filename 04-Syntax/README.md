# 04 - Syntax Library

## Why this folder exists

This folder is not a tutorial.

It is a quick reference for JavaScript syntax.

The goal is to reduce the time spent searching for syntax while solving problems.

Every file contains only syntax templates and short notes.

If you need to understand a concept, go back to its lesson in **01-Core-Javascript**, **02-DataStructures**, or **03-Patterns**.

Use this folder only when you already know *what* you want to do, but need a reminder of *how* to write it.

---

## What belongs here?

- Loop syntax
- Function syntax
- Array method templates
- Object syntax
- Set & Map syntax
- Promise syntax
- Async/Await syntax
- ES6 features

---

## What does NOT belong here?

- Long explanations
- Theory
- Interview questions
- Problem solving
- Diagrams
- Examples with detailed discussion

Those belong in the lesson folders.

---

## Goal

The Syntax Library should allow you to find the syntax you need within a few seconds without interrupting your problem-solving process.
