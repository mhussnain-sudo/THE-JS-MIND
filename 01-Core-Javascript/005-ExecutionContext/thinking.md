# Thinking

Forget JavaScript for a moment.

Imagine you're the JavaScript engine.

A file arrives.

Would you immediately execute line 1?

No.

First you prepare everything.

You create memory.

You remember function declarations.

You reserve space for variables.

Only then do you begin executing.

This is the mindset you should always have.

Whenever someone says:

"JavaScript starts."

Your brain should instantly think:

JavaScript

↓

Execution Context

↓

Memory Creation

↓

Execution

Not

JavaScript

↓

Execution

The preparation step is what explains Hoisting.
