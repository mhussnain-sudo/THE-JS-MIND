# 01 - Core JavaScript

## What is Core JavaScript?

Core JavaScript is the foundation of the language.

Everything else—React, Next.js, Node.js, Express, NestJS, Vue, Angular—builds on these concepts.

If the fundamentals are weak, frameworks become difficult to understand.

The purpose of this section is to build a deep understanding of how JavaScript works internally before using it to build applications.

## Goal

The objective is not to memorize JavaScript syntax.

The objective is to understand why JavaScript behaves the way it does.

Every concept should answer four questions:

1. What problem does it solve?

2. Why does JavaScript need it?

3. How does it work internally?

4. When should I use it?
