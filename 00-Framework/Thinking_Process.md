# Thinking Process

Every problem should pass through the same thinking process.

## T — Translate

Convert the programming question into simple English.

---

## H — Hunt

Look for keywords.

Examples:

Duplicate

Maximum

Minimum

Pair

Window

Unique

Reverse

Count

Sort

Exists

---

## I — Identify

Determine the problem category.

Examples:

Iteration

Lookup

Filtering

Transformation

Counting

Searching

Sorting

Aggregation

---

## N — Narrow

Choose the most appropriate approach.

Reject alternatives.

Explain why.

---

## K — Keep Building

Write an algorithm.

Dry run.

Only then write JavaScript.
