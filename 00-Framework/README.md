# 00 - Framework

## Why this folder exists

Before learning JavaScript, we must learn how to think.

Many developers struggle because they immediately jump into writing code. Experienced engineers spend most of their time understanding the problem before writing a solution.

This framework teaches a repeatable thinking process that will be used throughout THE JS MIND.

---

## Learning Order

1. Problem Solving Framework
2. Thinking Process
3. How To Read A Question
4. Decision Trees
5. Mental Checklist
6. Interview Approach

---

## Goal

The goal is to make problem solving systematic instead of based on guessing or memorization.

Every future lesson in this repository will follow this framework.
