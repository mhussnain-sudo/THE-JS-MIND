# Mental Checklist

Before writing any code, ask yourself:

□ What is the input?

□ What is the expected output?

□ What is changing?

□ What remains unchanged?

□ Which keywords stand out?

□ Which problem category does this belong to?

□ Which data structure fits best?

□ Which JavaScript feature should I use?

□ Which method fits naturally?

□ Can I explain my reasoning?

□ Can I dry run my solution?

□ What is the time complexity?

□ Can this solution be improved?
